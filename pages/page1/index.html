4 FUNDAMENTALS

Variables store data, while data types define its kind. Input and output enable user interaction. Conditional statements guide program flow, and loops handle repetitive tasks efficiently.


1. Variables and Data Types:

Variables store data values, while data types define the kind of data a variable holds.

2. Input and Output:

Input gathers data, and output displays it for interaction.

3. Conditional Statements:

Allows programs to make decisions based on conditions.

4. Looping Statements:

Repeats a block of code multiple times until a condition is met.
